RCC

Overview

    There are three types of reset, defined as system Reset, power Reset and backup domain Reset.
  • System reset:A system reset sets all registers to their reset values except the reset flags in the clock controller CSR register and the registers in the Backup domain
  • Software reset:The reset source can be identified by checking the reset flags in the RCC clock control & status register (RCC_CSR).
  • Power reset:A power reset is generated when one of the following events occurs: 1. Power-on/power-down reset (POR/PDR reset) or brownout (BOR) reset 2. When exiting the Standby mode
