OPAMP

Overview

    The STM32F30xx embeds four operational amplifiers that can either be used as standalone amplifiers or as follower / programmable gain amplifiers. The operational amplifier output is internally connected to an ADC channel for measurement purposes. OPAMP main features :
  • Rail-to-rail input/output
  • Low offset voltage
  • Capability of being configured as a standalone operational amplifier or as a programmable gain amplifier (PGA)
  • Access to all terminals
  • Input multiplexer on inverting and non-inverting input
  • Input multiplexer can be triggered by a timer and synchronized with a PWM signal.
