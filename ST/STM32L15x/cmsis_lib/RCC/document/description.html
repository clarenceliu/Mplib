RCC

Overview

  • The RCC provides a set of clock to drive the peripherals. Before use the peripheral, you should configure it clock source in RCC and enable it clock(except some continuous peripheral). NOTE: SYSCLK, HCLK and PCLK2 are set to 72MHZ, PCLK1 is set to 36MHZ by boot. The RCC component provides a set of APIs to configure system clock and peripherals clock.
