ADC

Overview

    ADCs main features:
  • High performance features
  • Low power features
  • External analog input channels for each of the 4 ADCs
  • In addition, there are seven internal dedicated channels
  • Start-of-conversion can be initiated
  • Conversion modes 每 Each ADC can convert a single channel or can scan a sequence of channels 每 Single mode converts selected inputs once per trigger 每 Continuous mode converts selected inputs continuously 每 Discontinuous mode
  • Dual ADC mode (on devices with 2 ADCs or more)
  • Interrupt generation at the end of conversion (regular or injected), end of sequence conversion (regular or injected), analog watchdog 1, 2 or 3 or overrun events
  • 3 Analog watchdogs per ADC
