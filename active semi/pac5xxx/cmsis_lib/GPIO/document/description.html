GPIO

Overview

  • Active semi pac5XXX Series has up to 37 General Purpose I/O pins can be shared with other function pins; it depends on the chip configuration. These 47 pins are arranged in 4 ports named with GPIOA, GPIOB, GPIOC , GPIOD and GPIOE. Each port equips maximum 8 pins. Each one of the 37 pins is independent and has the corresponding register bits to control the pin mode function and data.
